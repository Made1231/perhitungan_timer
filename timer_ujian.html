Panduan Penggunaan

1. Masukkan Waktu Total Ujian dan Waktu Peringatan dalam satuan menit.

2. Tekan tombol Mulai untuk memulai timer. Timer akan menunggu hingga suara pembuka selesai.

3. Gunakan tombol Stop jika ingin menghentikan sementara waktu.

4. Gunakan tombol Restart untuk mengatur ulang waktu sesuai input awal.

5. Gunakan tombol Percepat 1 Menit jika ingin mempercepat waktu mundur sebanyak 60 detik.

6. Aplikasi akan mengeluarkan suara peringatan secara otomatis sesuai dengan waktu peringatan yang diimput.

7. Saat waktu habis, aplikasi akan mengeluarkan suara akhir dan menghentikan timer.

8. Anda dapat mengganti waktu ujian kapan saja, lalu tekan Restart untuk mengatur ulang.

9. Pastikan suara pada perangkat aktif agar audio dapat terdengar dengan baik.

10. Untuk pengalaman terbaik, gunakan aplikasi ini dalam mode layar penuh.

Tentang Aplikasi

Aplikasi ini dikembangkan oleh Kelompok 1 dengan tujuan untuk mendukung pelaksanaan ujian OSCE (Objective Structured Clinical Examination) secara digital, efisien, dan akurat. Timer ini membantu penguji dan peserta untuk memantau waktu ujian dengan mudah, mengurangi kesalahan manusia, dan meningkatkan profesionalisme dalam pelaksanaan ujian klinis.

Proyek ini dibangun menggunakan teknologi web seperti HTML, CSS (Tailwind), dan JavaScript. Selain itu, desain responsif dan animasi interaktif juga diterapkan agar tampilan lebih menarik dan mudah digunakan di berbagai perangkat.

Dalam proses pengembangan, tim kami memfokuskan diri pada kemudahan penggunaan, kenyamanan visual, dan ketepatan waktu. Harapan kami, aplikasi ini dapat menjadi solusi sederhana namun bermanfaat untuk berbagai kegiatan ujian berbasis waktu, terutama di bidang pendidikan kesehatan.
